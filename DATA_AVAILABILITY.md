# Data availability

The data and code supporting the findings of this study are openly available in this GitHub repository.

The repository contains raw ZnO calibration traces, scripts for generating conventional and photonic-memory image sequences, scripts for adding Gaussian noise, reference training notebooks, the full experimental-grid configuration, and CSV result tables for the evaluated experimental conditions.

The full generated image datasets are not stored in the repository because they are deterministic intermediate outputs. They can be regenerated from the released scripts, random seeds/configuration, and calibration data.
